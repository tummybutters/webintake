# Harbor Peak Group Template Agent Guide

## Route
- `/templates/harbor-peak`

## Best Fit
- Firms that want a classic, stable, premium presentation
- Businesses using the site mainly for legitimacy and partner verification
- Capital, advisory, consulting, or acquisition groups that need clarity over flash

## Tone
- Calm
- Established
- Capable
- Broadly trustworthy

## Personalize These First
- Replace `Harbor Peak Group` with the actual business name
- Rewrite the hero section from the intake description
- Update service card headings and copy to fit the client's model
- Set the CTA to call, text, or email based on intake
- Insert real address, phone, email, and service footprint
- Add the client's brand colors if provided

## Guardrails
- Keep wording simple and credible
- Avoid niche jargon unless the client gave it directly
- Do not overstate experience, volume, or results
- Use this template when a more conservative presentation is the safest choice

## Design Notes
- This template works best with navy, slate, steel blue, sand, or neutral palettes
- Prefer clean copy blocks and limited decorative language
- Keep the premium feeling through spacing and typography, not extra effects

## Required Footer/Legal
- Add real contact info
- Add privacy policy language if form collection or SMS is present
- Include copyright line and legal entity name if provided
